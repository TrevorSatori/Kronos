# Kronos
Terminal Music Player Written In Rust. Used for offline listening and local files.

# State
Main branch is stable. features and perfomance improvements on the way.

# Controls
Enter => play
space => pause / resume
a => add song to queue
g => skip track
arrow keys / vim keys => navigate browser and queue
